# tru-sdk-react-native

React Native SDK for tru.ID

## Installation

```sh
npm install tru-sdk-react-native
```

## Usage

```js
import TruSdkReactNative from "tru-sdk-react-native";

// ...

const result = await TruSdkReactNative.multiply(3, 7);
```
